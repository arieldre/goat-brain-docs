Biggest Breakthroughs
The Council
Ambiguous decisions get four independent voices
When facing an architectural decision with no clear winner, don't ask one agent. Convene four — each with a different stance — spawned in parallel with minimal shared history to prevent anchoring. Disagreement is the output, not consensus.
Architect
Long-term correctness. What won't break in 6 months?
Skeptic
Challenge every assumption. What's the failure mode?
Pragmatist
What ships fastest with acceptable quality?
Critic
Edge cases and failure modes. What was missed?
Opus reviews all four, classifies findings P0/P1/P2, then proceeds. Used before any major architectural decision.
Semantic Memory
Relevant context arrives before you ask for it
Every session auto-injects the top 4 most relevant memories before Claude reads the first message. Score threshold 0.72. Context arrives silently. Every session starts smarter than the last.
Skill Files
Domain rules load when the keyword appears
Type "adset" — FB API gotchas load. Type "tROAS" — Google Ads rules load. No manual invocation. Zero overhead. Zero forgetting.

How It Compounds Over Time
WEEK 1
Seed
Migrate existing memory — game profiles, platform BPs, geo tiers, creative taxonomy. Shared namespace created in Pinecone. Every team member's Claude queries the same index.
MONTH 1–3
Accumulate
Each session auto-contributes. Saar tests a new UGC hook — lesson vectorized. Roy hits a SKAN timing gotcha — promoted to shared BPs. No duplication. Every observation becomes queryable.
MONTH 6+
Institutional
New hire queries "what creatives worked for UH US male 18–28?" and gets 6 months of team learnings instantly. The brain knows more than any individual. Onboarding time collapses.
